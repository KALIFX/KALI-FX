THE HARVEST TRADING SYSTEM

Overview and Summary of the Harvest Trading System

The Harvest Trading System is designed to filter entry signals using the Order Block indicator, which relies on price action interpretation. The ADX indicator is optionally included as a secondary filter. This strategy is best suited for high time frames.

Strategy Setup

  • Time Frame: 30 minutes or higher.
  • Currency Pairs: Any.

Indicators Used

  • Order Block: Default setting.
  • Harvest Entry Signal
  • ADX: Green line, 14 periods, level 20 (optional).

Trading Rules

Buy Signal:

  • Price bounces off support, triggering a buy signal.
  • ADX green line > 20 (optional).

Sell Signal:

  • Price bounces off resistance, triggering a sell signal.
  • ADX green line > 20 (optional).

Exit Strategies

  • Initial stop loss set above/below the support/resistance line or previous swing high/low.
  • Profit target with a minimum stop loss ratio of 1:1.3.

This strategy leverages the Order Block indicator for filtering trades and optionally uses the ADX indicator to confirm signals, ensuring trades are taken based on significant price action levels and trends.
